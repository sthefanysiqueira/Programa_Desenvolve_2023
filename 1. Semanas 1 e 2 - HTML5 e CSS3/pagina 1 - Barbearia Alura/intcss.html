    INLINE E BLOCK Block - A tag ocupa a largura inteira da página, bloqueando o conteúdo daquela linha. Inline - Permite que existam outros elementos na lateral. Uma imagem, por exemplo, não altera o conteúdo. Um elemento inline não permite alterar o espaçamento externo e interno dele. inline-block - Ocupa só o tamanho do conteúdo, mas permite mexer na largura e espaçamentos. É uma mistura das duas condições anteriores. ARRUMANDO A LISTA text-transform: uppercase - Transforma, visualmente, as letras de um texto em maiúsculas font-weight: bold; - É o peso da fonte. Bold define a fonte como negrito text-decoration: none; - Decoração de texto. Por exemplo, os sublinhados dos links. O none, está desativando estas decorações COMO FUNCIONA O POSICIONAMENTO position: static - posição padrão inicial position: relative - permite modificar a posição, espaçamento, etc. deslocar para os lados. position: absolute - Permite modificar o posicionamento absoluto em relação a outra coisa, por exemplo, em relação à pagina toda, ao cabeçalho, etc. AJUSTANDO O TAMANHO DOS ELEMENTOS box-sizing: border box - Define a largura como definitiva. Alterar a distância do espaçamento interno não irá mais afetar o espaço externo BORDAS ARREDONDADAS border-radius - Controla o raio da borda, permitindo tornar as bordas mais arredondadas. podem ser configurados todos os lados de uma vez apenas definindo o valor em pixels, ou de um por um escrevendo border-radius-top-left ou border-radius-bottom-right, por exemplo. PSEUDO-CLASSES DE ESTADO / COMPORTAMENTO DOS ELEMENTOS hover - controla o comportamento de um elemento quando se passa o mouse por cima dele. active - controla o comportamento de um elemento ao clicar ou pressinar ele. IMAGEM DE FUNDO background: url("bg.jpg") - define uma imagem como plano de fundo. Essa imagem ocupa todo o espaço do elemento. TRANSIÇÕES transition: cria um tempo de transição entre as mudanças definidas para quando o mouse está ou não sobre o objeto. cursor: Faz com que o ícone do mouse fique como um indicador sempre que passado por cima do objeto especificado. TRANSFORMAÇÕES transform: scale(); - Aumenta o tamanho do objeto proporcionalmente de acordo com o tamanho definido nos parênteses. transform: rotate(xdeg); - realiza uma rotação do objeto, de acordo com os graus informados nos parênteses. SELETORES AVANÇADOS main > p { }; - Permite fazer modificações nos parágrafos que são filhos diretos da main, ignorando outras seções e divisões. img + p { }; - Permite selecionar o primeiro parágrafo que vem sucede o elemento 1, no caso, uma imagem. img ~ p { } - Permite modificar todos os parágrafos que vem depois do elemento no um, no caso, todos os parágrafos que vem depois de uma imagem. .principal p:not(#missao) { background: orange; } - Permite excluir itens especificos. No caso, exclui todos os parágrafos que não compõe missao. CÁLCULOS COM CSS calc(x - y) Tira y de x. Também podem ser feitas outras contas e outras operações. Ex: width: calc(40% - ((26px * 4) / 4)); OPACIDADE NOS ELEMENTOS opacity: Permite controlar a opacidade das imagens e elementos. Vai de 0 a 100%, ou seja, de 0 a 1. EX: opacity: 0.3; rgba: adiciona uma camada de opacidade nas cores RGB, chamada de "alfa". Esse recurso pode ser utilizado em todas as cores: de fundo, texto, borda e assim por diante. EX: color: rgba(0,0,0,0.3); SOMBRA NOS ELEMENTOS EXTERNA: box-shadow: X Y #; Possui as propriedades do eixo X, T e uma cor. Ex: box-shadow: 10px 10px #000000; É possível ainda, adicionar uma terceira propriedade chamada blur, que aplica um nível de desfoque na sombra. No exemplo, são aplicados 5px. Ex: box-shadow: 10px 10px 5px #000000; Também há uma quarta propriedade que configura o tamanho da sombra projetada. No exemplo, são inseridos 20px: box-shadow: 10px 10px 5px 20px #000000; É possível ainda, inserir outras sombras com a separação através de uma vírgula (,). Podendo inclui, até mesmo sombras com cor rgba(). Ex: box-shadow: 10px 10px 5px 20px #000000, -10px -10px yellow, -20px 20px rgba(255,0,0,0.5); SOMBRA INTERNA: Feita com o uso do "inset". Para que se torne perceptível na página, é preciso de um espaçamento. No exemplo, são inseridos 30px. Ex: box-shadow: inset 0 0 30px #FF0000; OBS: Sombras internas não são deslocáveis.
